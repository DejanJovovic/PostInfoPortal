Cross-platform mobile news application built with React Native and Expo that provides access to posts from postinfo.rs using Wordpress REST API. Main features of the application are: filtering posts by categories/subcategories, offline caching, favourite posts, advanced search/date filters, sorting, push notifications with OneSignal, sharing posts, light/dark themes, stack navigation, deep linking and a clean modern UI with NativeWind.
Officially released on Google Playstore:  https://play.google.com/store/apps/details?id=com.gosmartdevelopment.postinfoportal  
Test iOs version live.
